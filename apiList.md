# DevTinder APIs

## authRouter
- post/signup
- post/login
- post/logout

## profileRouter
- get/profile/view
- patch/profile/edit
- patch/profile/password // forgot password API

## RequestRouter
- POST /request/send/status/:userId
- POST /request/review/status/:requestId


## userRouter
- GET /user/connections
- GET /user/requests
- GET /user /feed - Gets you the profiles of other users on platform
